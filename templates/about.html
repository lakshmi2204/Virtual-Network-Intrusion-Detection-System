About VNIDS

VNIDS (Virtual Networking Detection System) is a cybersecurity monitoring platform designed to detect suspicious activites and network attacks in real time.

The system monitors live network traffic, detects malicious behaviour, stores attack logs, and generates alerts for security analysis.

Features

  • Real-time Packet Monitoring
  • Port Scan Detection
  • SYN Flood Detection
  • Severity Classification
  • SQLite Log Storage
  • Real-Time Alert System
  • Interactive Security Dashboard

Technologies Used

  • Python
  • Flask
  • SQLite
  • Scapy
  • Socket Programming
  • HTML/CSS/JavaScript
  • Chart.js
  • VMware Linux & Ubuntu
